About AShifter

I'm AShifter.

Hello! I'm AShifter. I make things and put them on the internet, Including this website. This website showcases a few interesting projects I've been playing with. I'm extremely passionate about the work I do!

MonoHack

MonoHack is an in-progress game engine built on MonoGame for use with hacking games and general game UI needs. It was born from the ashes of the old ShiftOS project from 2014 and will hopefully live on to be the most versatile UI toolkit for MonoGame and XNA-based projects.

WaterBot

WaterBot is a Discord Bot. Simple! It's completely modular, and gets improved whenever I have the time to work on it. It can't compete with the likes of AstralMod and PrecipitationJS (yet!) but I hope to get it there some day.
