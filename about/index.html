About AShifter

I'm AShifter.

Hello! I'm AShifter. I develop things and put them on the internet. Including this website!

This website showcases my best GitHub projects. The three tiles on my homepage (which can be accessed from any page by clicking the "ashifter" text at the top) are the projects that I love to devote time to and experiment with. Clicking on any of the tiles will bring you to that project's page which will present you with a GitHub README.md file which is grabbed from that repository and parsed into HTML with the Showdown Markdown Parser. Clicking on the README.md text in any project page will bring you to the GitHub repository where you can clone, fork, and contribute.

MonoHack

MonoHack is an in-progress game engine built on MonoGame for use with hacking games and general game UI needs. It was born from the ashes of the old ShiftOS project from 2014 and will hopefully live on to be the most versatile UI toolkit for MonoGame and XNA-based projects.

WaterBot

WaterBot is a Discord Bot. Simple! It's completely modular, and gets improved whenever I have the time to work on it. It can't compete with the likes of AstralMod and PrecipitationJS (yet!) but I hope to get it there some day.
